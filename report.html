How to report issues?

If the kernel not work well you can report me the issues with log/kmsg.
First, before reporting any problems make sure it is a kernel problem, and you have installed last version of Rave kernel from official download link.

With phone booted and root:

  • Use FK Manager for export Kmsg (ask in support group for apk);
  • Use "adb logcat | tee log.txt" from PC and send log.txt file.

With phone not booted or without root:

  • Go to recovery and take /proc/last_kmsg file;

If Phone booted:

  • Take also ROM log with "adb logcat | tee log.txt" and send log.txt file.

When you report a problem make sure you are willing to take a possible test to help me solve it.

Common issues (lavender):

  • Lavender ROM use also QTI drivers. Flash right kernel version;
  • Some lavender ROM support linux 4.19.y and Rave kernel for lavender is linux 4.4.y;
  • If camera not work make sure you have installed right version of camera (oldcam or newcam). QTI is oldcam.
