How to report issues?

If you find problems remember to follow this guide.

Get the log

To understand the cause of the problems I need to see the debug log. Can:

  • Get the kernel logs via FrancoKernel Manager
  • Use MatLog
If the problem is the device not booting, check that you have installed the correct version. After doing this:
  • Using ADB:
    1. adb devices
    2. adb logcat | tee log1.txt
  • Get the file /proc/last_kmsg from recovery

After you get these files, post them to my support group specifying your problem.
Without the logs any problems will be ignored.
