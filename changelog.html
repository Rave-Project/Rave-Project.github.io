Feature

Here you can see the changelogs for the different devices.

Lavender

Added support for Android 12, imported patches needed for boot and full operation. I had to rebase from previous versions to better update the base, compiled with Proton Clang.

Last beta release:

  • Backport bpf
  • Backport binder from linux 5.4
  • Disable Kernel Page Table Isolation
  • Use LA.UM.7.2.r1-09400-sdm660.0 staging drivers
  • Upstream Linux v4.4.296

Begonia

Based on begonia-q-oss support firmware 12.0.x. Import some caf patches as upstream of the memory management drivers, adjustments in the 850 Mhz GPU and OC drivers. Disabled outdated security systems and other unused drivers, increase performance in critical cases. Using SLMK as RAM manager. Compiled with Eva GCC toolchain.

Last relese:

  • Update module blocker
  • Tweak scheduler
  • Disable integrity security
  • Enable ZRAM Writeback & optimize
  • Upstream Linux v4.14.246
